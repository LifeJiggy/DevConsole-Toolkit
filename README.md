‎Quick Start
‎
‎In Chrome DevTools
‎
‎Open DevTools (F12 or Cmd+Option+I).

‎Navigate to the Sources tab > Snippets panel.
‎
‎Create a new snippet, paste the code from any tool below, and hit Run (Ctrl+Enter).
‎
‎Save snippets for reuse—boom, instant toolkit!
‎
